04 / 08

Local-first

Data stays local, keys never leave

$ git clone github.com/nexu-io/open-design

# .od/ — all runtime data lives on your machine
.od/
  app.sqlite          # local SQLite, no cloud dependency
  projects/           # agent working directories
  artifacts/          # saved render artifacts
  media-config.json   # BYOK credentials, never uploaded

byok:
  providers: [anthropic, openai, azure, google]
  self_host: true     # Apache-2.0, self-host freely
