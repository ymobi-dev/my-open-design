Our Approach
Workflow 06
How It Works

From idea to delivery in five steps

Each step feeds the next: every choice becomes context for the run after it.

  • Pick a starting point — describe your goal in one sentence, or start from a template or plugin
  • Set the visual direction — palette, type, and spacing flow into generation automatically
  • Generate the artifact — the agent reads full context and produces real runnable files, previewed in a sandbox
  • Ship it, or film it — hand off to engineering, or turn it into a marketing video with HyperFrames
  • Memory compounds — every choice becomes context for the next run

The Shift 08
Closed & Cloud-only

One vendor, one model — design locked in the cloud

Closed-source and paid. The model is fixed, agents cannot be swapped, and self-hosting and BYOK are off the table.

  • No agent choice
  • No self-hosting, no BYOK
  • Your design context lives on someone else's servers
Open Design

Open, local, and agent-native

Apache-2.0 and runs on your machine. Bring your own keys, plug in the agents you already use, and keep every artifact as real files.

  • 21 coding agents, zero config
  • Full-stack BYOK — Anthropic / OpenAI / Azure / Google
  • -80% monthly cost vs. Claude Design ($500 → ~$73)
