The insight

AI can truly design now — it should be open

In April 2026, Claude Design proved an LLM can produce real design artifacts. But it's closed, paid and cloud-only. Open Design rebuilds it in the open.

  • Desktop-native: local files, Figma, repos
  • Plugs in Claude Code, Codex, Cursor
  • Memory compounds with every choice
  • Apache-2.0 — fork, self-host, go commercial

The workflow

Idea to delivery in four moves

Pick

Describe your goal in one sentence, or start from a template or plugin.

Style

Set the visual direction — palette, type and spacing flow into generation.

Generate

The agent produces real runnable files, previewed and edited in a sandbox.

Ship

Hand off to engineering or film it with HyperFrames — every choice becomes memory.
